--------------------------------------------------------------------------
PvpScribe

AIE PvP Statistics Add-On for World of Warcraft

Author: Mark Rogaski <stigg@pobox.com>
--------------------------------------------------------------------------

--------------------------------------------------------------------------
Installation
--------------------------------------------------------------------------

To install this add-on, unzip the contents of the distribution ZIP file
to the add-ons directory of your World of Warcraft installation.

[WoW Install Dir]\Interface\AddOns

The unzipping process should create a directory with the following name.

[WoW Install Dir]\Interface\AddOns\PvpScribe


--------------------------------------------------------------------------
Usage
--------------------------------------------------------------------------

/pvpscribe { show | clear | history | standby | about | debug }

/pvps { show | clear | history | standby | about | debug }


Commands:

show        Display the current summary of PvP statistics.

clear       Clear the database of PvP statistics.

history     Set the maximum number of battleground sessions that complete
            records are kept for.  Not that this does not affect summary 
            statistics.

standby     Enable or disable the add-on.

about       Display information about the add-on.

debug       Toggle Ace2 debugging.  Currently, this does nothing of use.
